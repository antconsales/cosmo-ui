# Web Basic Example

Basic Next.js example demonstrating Aura UI HUDCard component.

## Getting Started

```bash
# From the root of the monorepo
pnpm install
pnpm build

# Then run this example
cd examples/web-basic
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the example.

## What's Demonstrated

- Basic HUDCard rendering
- Card actions and dismissal
- Integration with @aura/core-schema and @aura/renderer-web
